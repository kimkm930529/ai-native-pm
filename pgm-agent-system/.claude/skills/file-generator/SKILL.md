# file-generator — Skill Spec

## 역할

`output/analysed_report.json`과 `config.json`의 템플릿을 결합하여
로컬 마크다운 파일(`output/flash_{YYYYMMDD}.md`)을 생성한다.

## 호출 방법

```bash
python3 .claude/skills/file-generator/scripts/generate_md.py \
  --input output/analysed_report.json \
  --template config.json \
  --output output/flash_{YYYYMMDD}.md
```

## 출력 파일 구조

생성되는 마크다운 파일 구조:

```markdown
# Weekly Flash — YYYY년 MM월 DD일 주간 보고

> 완료 **N건** (SP: **N**) | 진행 중 N건 | 블로킹 N건

---

## ✅ 성과 (총 N SP 완료)

- ⭐ **[MATCH-123] 티켓 제목** — highlight 요약
- [MATCH-124] 티켓 제목 — highlight 요약

## ▶ 진행 현황

- ▶ **[MATCH-125] 티켓 제목** — progress_note
- [MATCH-126] 티켓 제목

## 📋 다음 주 계획

- **[MATCH-127] 티켓 제목**
- [MATCH-128] 티켓 제목

## 🚫 이슈 / 블로킹

- 🚫 **[MATCH-129] 티켓 제목** — block_reason

---
*Auto-generated by pgm-agent-system — YYYY-MM-DD*
```
