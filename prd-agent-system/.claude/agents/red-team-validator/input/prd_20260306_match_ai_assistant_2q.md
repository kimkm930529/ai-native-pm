# MATCH AI Assistant 마케팅 인사이트 고도화 — 2Q PoC 확대

## 배경

사용자가 MATCH 내 AI Assistant를 통해 마케팅 인사이트와 관련 정보를 효율적으로 획득할 수 있도록 지원한다.

## As-is (현황)

Phase 1. 1차 PoC 결과 피드백: PAGE0 등 메트릭 학습 범위 내 하이레벨 분석 관점 유의미한 유스케이스 확인

유의미한 유스케이스 예시:
- 무신사 플랫폼 신규 구매자, 신규 가입자가 25년 12월 말 대비 26년 1월 3-4일까지 크게 뛰었다가, 26년 1월 5-6일에 급감했는데 원인이 뭐야?
- 무신사 고활성 고객의 감소가 어느 시점부터 일어났는지, 그 수준이 어느 정도인지 지난 1년 시계열 트렌드를 기반으로 분석해줘

## To-be (개선)

1Q PoC 활용 방향성 확대하기 위해 2Q 스콥 진행

- 오디언스 확대: 마케팅 도메인 실행팀 직원
- 분석 범위 확장: POC의 Page0(하이레벨 뷰)를 넘어 마케팅 도메인 마트를 학습 범위로 스코프 확대

## 기대효과 (Success Criteria)

- Quarterly 기여 GMV 64억 예상 (퍼포먼스/CRM 마케팅 운영효율 +10% 개선효과 기대, 연 260억 광고예산 * ROAS 2000% 기준 임팩트 산정)
- 분석 리드타임 단축: 마케팅 분석에 소요되는 시간을 최소 50% 이상 단축 기대함 (→ 연 0.8억 운영비 효율화)

## 희망 런칭 일정 및 사유

- 25'Q4 POC에서 정의된 실행/분석의 방향을 구체화하여 2분기 정식 과제로 전환
- 핵심: DI 조직과의 L1/L2 마트 구축 협력을 통해 분석 정확도 확보
- 29CM 우선 집중 전략 (SL 이니셔티브 진척 타임라인에 따라 무신사 3Q 과제로 조정)
