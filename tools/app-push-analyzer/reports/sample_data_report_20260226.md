## 2. 잘된 점 / 개선이 필요한 점

### ✅ 잘된 점

#### A. 행동 기반 타겟팅 캠페인의 압도적 ROI

```
┌─────────────────────────┬──────────┬────────┬────────┬──────────────┐
│ 캠페인                   │ 발송수    │ CTR    │ 전환율  │ 발송당매출    │
├─────────────────────────┼──────────┼────────┼────────┼──────────────┤
│ 🥇 장바구니 리타겟팅      │  30,000  │ 40.0%  │ 30.0%  │   806.67원   │
│ 🥈 재구매 유도 쿠폰       │  20,000  │ 40.0%  │ 30.0%  │   691.00원   │
│ 🥉 신상 한정수량 알림     │  45,000  │ 35.4%  │ 25.0%  │   600.00원   │
├─────────────────────────┼──────────┼────────┼────────┼──────────────┤
│    전체 평균              │  74,583  │ 29.3%  │ 25.7%  │   336.57원   │
└─────────────────────────┴──────────┴────────┴────────┴──────────────┘
```

- **장바구니 리타겟팅**은 발송당매출 **806.67원**으로 전체 평균의 **2.4배**
- **재구매 유도 쿠폰**은 발송수 20,000건으로 최소 규모임에도 매출 **1,382만원** 달성
- 공통점: **CTR 40%** 달성 → 개인화된 메시지가 행동 전환을 효과적으로 유도

#### B. 긴급성·희소성 메시지의 높은 오픈율

| 캠페인 | 오픈율 | 전체 대비 |
|--------|--------|-----------|

| 플래시 세일 30분 | **30.0%** | +4.56%p |

> "한정", "30분", "장바구니 남은 상품" 등 **긴급성 키워드**가 포함된 푸시가 오픈율에서 명확한 우위

#### C. 월 후반부(2/14~2/26) 성과 집중

### ⚠️ 개선이 필요한 점

#### A. 대량 발송 캠페인의 심각한 효율 저하

```
┌─────────────────────────┬──────────┬────────┬────────┬──────────────┐
│ 캠페인                   │ 발송수    │ 오픈율  │ 전환율  │ 발송당매출    │
├─────────────────────────┼──────────┼────────┼────────┼──────────────┤
│ ❌ 주간 특가 세일         │ 120,000  │ 20.0%  │ 15.0%  │    72.00원   │
│ ❌ 봄 컬렉션 프리뷰       │ 100,000  │ 20.0%  │ 15.0%  │   144.00원   │
│    (2/21 미확인 캠페인)   │  80,000  │ 18.0%  │ 15.0%  │    64.75원   │
├─────────────────────────┼──────────┼────────┼────────┼──────────────┤
│    전체 평균              │  74,583  │ 25.4%  │ 25.7%  │   336.57원   │
└─────────────────────────┴──────────┴────────┴────────┴──────────────┘
```

- 이 3건은 전체 발송의 **33.5%**(30만건)를 차지하지만 매출 기여는 **10.7%**(3,222만원)에 불과
- 오픈율 18~20%, 전환율 15%로 **전체 평균 대비 오픈율 -5~7%p, 전환율 -10%p**
- **발송당매출 64.75~144원** → 최고 효율 캠페인 대비 **1/5~1/12 수준**

> 💡 **핵심 문제:** 비타겟팅 대량 발송이 전체 효율을 끌어내리고 있으며, 장기적으로 푸시 수신 거부율 증가 리스크

#### B. 멤버십 혜택 안내의 클릭→전환 병목

| 지표 | 멤버십 혜택 안내 | 전체 평균 | 차이 |
|------|----------------|-----------|------|
| 오픈율 | 30.0% | 25.44% | +4.56%p ✅ |
| CTR | **20.0%** | 29.28% | **-9.28%p** ❌ |
| 전환율 | **19.98%** | 25.66% | **-5.68%p** ❌ |

- 오픈은 잘 되지만 **클릭 단계에서 대량 이탈** → 랜딩 페이지 또는 CTA 메시지 개선 필요
- 수신율 98.0%로 최고 수준 → 세그먼트 품질은 좋으나 **혜택의 구체성이 부족**한 것으로 추정
